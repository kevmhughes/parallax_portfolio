DALL•E 2

This application is a full stack project built using the MERN stack. It is a clone of OpenAI's DALL•E 2 app, built using the following: MongoDB, ExpressJS, ReactJS, NodeJS, Tailwind CSS, the OpenAI API, and Cloudinary. Users are able to: create imaginative and visually stunning images through DALL•E 2 AI. browse through a collection of images generated by DALL•E 2 AI. share the generated imags with the community. download the generated and shared images. view details of the shared images.
